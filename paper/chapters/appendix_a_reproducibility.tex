\chapter{Reproducibility Notes}

\section{Software Dependencies}

The simulator depends only on \texttt{numpy} and \texttt{matplotlib}. The main experiment entry point is \texttt{main.py}, which calls \texttt{run\_all\_experiments()} from \texttt{project/experiments/run\_experiments.py}. The paper figures used here are generated separately in the \texttt{paper/} directory from the same underlying numerical model.

\section{Deterministic Execution}

All bundled experiments are deterministic. There is no stochastic sampling, no random seed, and no Monte Carlo component in the current engine. Re-running the project reproduces the same CSV values up to ordinary floating-point precision.
